2.1.2. Instructor Interface Overview

Navigation Bar

The instructor interface consists of a navigation bar at the top of the screen, containing a customizable picture for their user account and two tabs (Preferences and View Schedule) on the left, and a schedule selector dropdown menu and an options button on the right.



Figure 2.1.5. Instructor Interface Welcome Screen

Before the instructor selects one of the tabs, the same two options are shown in the center of the screen, underneath a friendly welcome message.

Editing Preferences

When the instructor clicks either the Preferences tab on the navigation bar or the shortcut presented on the welcome screen, the system creates a Preferences form and populates the fields with the current values stored in the database.

This form comprises the following elements:

  1. WTU Panel - a panel on the upper left to edit the number of Work Time Units (WTUs) they are going to teach
  2. Course List - a list of courses they are eligible to teach
  3. Course Specific Preferences Panel - contains a visual set of tools needed to teach the course, a calendar widget to specify when they would prefer to teach it, and a slider to determine how much they want to teach it
