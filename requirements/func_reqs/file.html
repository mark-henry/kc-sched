2.6. Details of File Commands

The scenarios in this section cover full details of file-related commands.

2.6.1. New and Open

When the user executes the ''New' command in the `File' menu, the system responds by creating a new Schedule and displaying its initial view.

When the user selects the 'Open' command in the `File menu, the user may type the name of the file to open or select it from the scrollable list of file names.

2.6.2. Save and Save As

When the user selects the 'Save' in the 'File' menu, the Schedule will be automatically saved to its specified file. If the file is new and unsaved, a dialog will appear asking the user where to save the file. Selecting 'Save As' has the same effect as clicking 'Save' for the first time.

2.6.3. Export

When the user selects 'Export' in the 'File' menu, the Export tab in the program will be displayed, giving options for the user to export to file.

2.6.4. Print

Printing from the Scheduling Tool is based strictly on the content visible in active Scheduling Tool windows. Window content appears on the printed page exactly as it appears on the screen.
