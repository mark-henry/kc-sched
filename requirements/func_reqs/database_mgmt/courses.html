2.2.2. Course Sub-Tab

Adding a Course

Click the plus button in the Courses Added part of the screen to enter new course information.

When adding or editing a course:
  1. Course Name is the name of the course contained in the course catalog.
  2. The Course Number field must contain only numerical values.
  3. The Lab checkbox may be checked or unchecked.
  4. Department contains the name of the department the course is listed under.
  5. Units are the number of units the course satisfies.
  6. Comments is used for any additional comments the scheduler wishes to add for the course.
To edit an existing course in the list, the user clicks on a course and all the related information is filled in.  The user can then edit any of the information and click the "Commit Changes" button to save the room with the new information.  If the user would like to cancel any edits made on a course they can click the "Restore" button to revert back to the last saved state.
