1.5.? aSc Timetables

 

aSc Timetables is a scheduling application targeted at primary/secondary schools. They maintain two separate versions of the application:

 

  • •asc Timetables – Microsoft Windows application that closely mimics the UI of Microsoft Office (including the ribbon in recent releases) 

  • •asc Timetables Online – Web application hosted on aSc servers which enables sharing and input via password-protected user accounts for teachers, online backups, and many other features not included in the desktop version 

 

Good Features:

 

  • –Intuitive and easy-to-learn interface, plus comprehensive help system 

  • –Wide variety of constraints can be applied to schedules fairly easily 

  • –Classes, teachers, courses, etc. can be assigned colors to make the schedule easy to read 

  • –Synchronization with school database software 

  • –Interactive (schedule can be tweaked once generated) 

  • –Imports from XML 

  • –Exports to XML, HTML, Excel, etc. 

 

Bad Features:

 

  • –Designed for primary/secondary schools; may not be ideal for university 

 

Missing:

 

  • –Desktop version missing most features listed above 

  • –Desktop version only runs on Microsoft Windows
