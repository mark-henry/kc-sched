2.1. User Interface Overview

Old Versions:

Login Screen (v1)
Welcome Screen (v1)

When the user opens the application, they are presented with a login screen:

The background of the login screen as well as the department logo is configured by the administrator (scheduler), so that the instructors are greeted with a familiar screen.
The nofifications window on the right informs them of any deadlines or other useful information


After logging in, the user is presented with an interface customized just for them.

The Instructor Welcome Screen gives the user a view with options to (a) view the current schedule or past schedules and (b) edit their preferences to supply the scheduler with constraints to feed to the scheduling algorithm.

The Administrator (Scheduler) Welcome Screen gives the user a similar layout as the instructor, but with all the features available to generate, edit, and publish schedules.
