2. Functional Requirements

The primary functionality the Scheduler system is for generating a schedule of courses for a university term. Scheduling operations are performed by administrative users, who schedule courses that are associated with professors. The Calendar system also provides functionality for users to set a variety of scheduling preferences.

Following an overview of the Scheduler user interface, details of system use are presented in the following scenarios:

  • Administrator Interface
    • Administrator Welcome
    • Add / Edit Instructor
    • Add / Edit Course
    • Add / Edit Room
    • Generate Schedule
    • View / Edit Generated Schedule
    • Export Schedule
  • Instructor Interface
    • Instructor Welcome
    • Add / Edit Instructor Preferences
