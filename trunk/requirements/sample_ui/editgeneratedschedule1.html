2.1. User Interface Overview

Ui scenario that greets the user after a schedule is generated and selected for viewing.

Key features:

-          Can easily view previous quarters schedules, or filter by a specific term(ie. fall, winter spring)

-          Can drag/drop a course from one time to another. Algorithm calculates for conflicts, and prompts user with solution(s).

-          Very snappy and smooth feeling

-          Down arrow to see later times in the day
