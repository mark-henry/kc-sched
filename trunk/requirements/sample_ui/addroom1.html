2.1. User Interface Overview

Room profile view. From here you can add/edit a room profile.

Key Features:

-          Easy to navigate side bar and top bar

-          All information is clearly shown, with features that are used often are displayed large on the screen

-          Room accessories easy to view and edit

-          Preferences button hides all user configurable items (ie. do not show room location, do not show rooms not in possession by your department, etc.)

-          + button adds another room, or service to the room, - button takes away item in room. To delete room must use preferences or options button
