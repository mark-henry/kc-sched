2.2.2 Add / Edit Instructor

Instructor Tab Overview

This is a view that is available only to a system admin. It will allow the admin to see all of the teachers, and their related info. The window on the left contains every instructor's name that is in the database, sorted in alphabetical order (by last name). When selecting an instructor on the left, it becomes highlighted (like we see below for "Bellardo, John"). When selecting an instructor, the window on the right displays all that professors information. All information is displayed in editable text-fields.

Modifying an Instructor

To modify an instructor's information, in this case professor Bellardo's information, simply select Bellardo from the left window to pull up his information. Once his information is displayed in the right window, all the text-fields are editable. One can simply click inside the phone number field to change / modify. Once modifications are made, a user can "commit changes" to write the new values to the database, or "restore" the text fields to the previous values.

Each field has the following constraints:

  1. First Name: Must contain ONLY alphabetical characters (a-z and A-Z). Limited to 30 characters.
  2. Last Name: Same as first name.
  3. Phone Number: Must contain ONLY numerical characters and the dash character. (1-9 and '-') Limited to 12 characters.
  4. Office Number: Can contain alphanumeric characters. Limited to 10 characters.
  5. Department: Must contain ONLY alphabetical characters. Limited to 100 characters.
  6. Email Address: Must be a valid email address, and properly formatted (name@domain.com)
**Any violation of these restrictions results in the error/ alert message shown in figure ... The alert is triggered when the user clicks off the text-field they just modified. After the error message is viewed and the user clicks "ok," the text-field in violation will have the cursor inside, ready to edit.

Adding an Instructor

To add an instructor, one would click the "+" button at the bottom of the left window, as shown in figure 6. Once the user clicks this button, the window on the right will display a blank set of information corresponding the new instuctor just added. One would fill these text-fields in as appropriate, and when finished, one would click the "commit changes" button to update the information of the new instructor. Once a user clicks the "commit changes" button, the instructor name will update to the new name. Clicking "restore" in this case, will undo the add.
