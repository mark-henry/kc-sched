Lantiv Timetabler


Good:

--contains multiple databases to handle what teachers, rooms, subjects, etc are available

--allows both interactive and automatic schedule creation

--during interactive mode, warnings are triggered if conflicts are detected

--able to highlight any number of days to see them all laid out in a calendar view

--able to get a server version that can be accessed remotely through flash enabled browser to allow simultaneous editing


Bad:

--very poor calendar view above ~5 days, beyond that just squishes all the info into the same amount of space

--can only import text, XML, and MS Access files

--very small/poor window for viewing schedule of an individual


Missing:

--no export feature, expected to be the final medium for creating and viewing schedules
