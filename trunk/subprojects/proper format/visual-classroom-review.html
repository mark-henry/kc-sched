Visual Classroom Scheduler


Visual Classroom Scheduler (VCS) is room scheduling software that allows for a user to quickly an easily create class schedules for campuses. The features that this tool provides allows the user to streamline the process by which they make schedules. It allows the importing and exporting files for quick creation or changing of schedules.


VCS provides an acceptable level of scheduling functionality. It only supports two file formats which are widely used for other pieces of software.


Good Features:

  • a simple graphical interface that users can figure out quickly
  • visual representation of schedule with color coding for classrooms
  • importing and exporting of files for sharing with other applications
  • shows conflicting items in a schedule
  • easy to change schedules to correct errors in schedule
  • can archive previous schedules
  • can use previous schedules as a start point


Bad Features:

  • import/export only two file formats (html not one)


Missing Features:

  • generation of schedules using provided databases
  • does not interface with a database management systems
  • does not allow for preferences to be noted in professor entries
  • does not take into consideration course offerings in a term
